# Ian Coyle

**Era:** W+K Portland (c.2010–2011); King/Coyle (2010–2015)

## Overview

Ian Coyle is an interaction designer credited, alongside Duane King, as the primary creator of *Nike Better World*'s famous parallax scrolling effect — the technique that made the site the most discussed website of 2011 and established a new paradigm for web design.

Multiple sources identify Ian Coyle as the originator of parallax scrolling in mainstream web advertising:

- **GWS Media:** *"Its popularity started to rise significantly in 2011, which was around the time Ian Coyle created the very first parallax website for Nike, 'Nike Better World'."*

Following the Nike Better World project, Ian Coyle co-founded **King/Coyle** with Duane King, a studio that continued developing the interaction language they pioneered at W+K. He also created a widely-used Parallax.js plugin.

## Key Projects

- **[Nike Better World](../projects/wk_nike_better_world.md)** — Design and Interaction co-lead (2011). Co-created the parallax scrolling effect that became one of the most influential web design techniques of the decade.
